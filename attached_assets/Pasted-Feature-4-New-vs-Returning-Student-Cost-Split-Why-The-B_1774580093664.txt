Feature 4: New vs. Returning Student Cost Split
Why: The Bellwether model budgets several costs differently for new vs. returning students. "Textbooks (new): $500/new child" vs. "$100/existing child." "Non-instructional equipment (new): $76/new child" vs. "$25/existing child." This is financially significant — Year 1 of a startup (all new students) costs 3-5x more per student for these categories than Year 4 (mostly returning). SchoolStack collects retentionRate but only uses it for display and coaching — never for financial calculations.
Current state:

enrollment.retentionRate collected in EnrollmentStep (line 168+), stored in schema
Used only in DemandConfidenceIndicator (display) and consultant engine (coaching text)
Revenue and expense calculations use total enrollment — no new/returning distinction
Expense driver types: annual_fixed, monthly, per_student, percent_of_revenue

Computation Changes
File: artifacts/api-server/src/lib/workbook-helpers.ts

Add enrollment split helpers:

typescriptexport function computeNewStudents(
  enrollment: number[], retentionRate: number, y: number
): number {
  if (y === 0) return enrollment[0]; // Year 1: all new
  const returning = Math.round(enrollment[y - 1] * (retentionRate / 100));
  return Math.max(0, enrollment[y] - returning);
}

export function computeReturningStudents(
  enrollment: number[], retentionRate: number, y: number
): number {
  if (y === 0) return 0; // Year 1: none returning
  return Math.min(enrollment[y], Math.round(enrollment[y - 1] * (retentionRate / 100)));
}

Update driverVal() (line ~308) to handle new driver types:

typescriptexport function driverVal(
  amounts: number[] | undefined, y: number, dt: string,
  students: number, escalationRate?: number, fallbackInflation?: number,
  newStudents?: number, returningStudents?: number  // NEW
): number {
  let base = amounts?.[y] ?? 0;
  const esc = resolveEsc(escalationRate, fallbackInflation);
  if (esc !== 0 && y > 0) {
    const y1 = amounts?.[0] ?? 0;
    base = y1 * Math.pow(1 + esc / 100, y);
  }
  switch (dt) {
    case "monthly": return base * 12;
    case "per_student": return base * students;
    case "per_new_student": return base * (newStudents ?? students);
    case "per_returning_student": return base * (returningStudents ?? 0);
    case "annual_fixed": return base;
    default: return base;
  }
}
