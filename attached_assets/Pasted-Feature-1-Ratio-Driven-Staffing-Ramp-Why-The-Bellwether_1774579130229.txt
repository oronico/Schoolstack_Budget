Feature 1: Ratio-Driven Staffing Ramp
Why: The Bellwether model's core innovation is enrollment-driven staffing. Every position has a trigger ratio — "22 students per teacher," "200 students per SPED teacher," "550 students per secretary." As enrollment grows from 88 to 400 students, FTEs auto-calculate from 4 to 18 teachers. SchoolStack's static staffing roster (same people, same FTEs, all 5 years) fails the lender sniff test for any school projecting enrollment growth.
Current state:

StaffingRow has a fixed fte field (0.0–1.0 per row)
computePersonnelForYear() in workbook-helpers.ts (line 605) applies the same fte to all 5 years, only varying by salary escalation and Year 1 proration
staffingRowSchema in schema.ts has no ratio or year-range fields

Add a new helper function:

typescript/** Compute effective FTE for a staffing row in a given year */
export function computeEffectiveFte(r: StaffingRow, y: number, enrollment: number): number {
  // Year range check (y is 0-indexed, startYear/endYear are 1-indexed)
  if (r.startYear && (y + 1) < r.startYear) return 0;
  if (r.endYear && (y + 1) > r.endYear) return 0;

  if (r.staffingMode === "ratio" && r.studentRatio && r.studentRatio > 0) {
    let computed = enrollment / r.studentRatio;
    // Apply floor/cap
    if (r.minFte !== undefined) computed = Math.max(computed, r.minFte);
    if (r.maxFte !== undefined) computed = Math.min(computed, r.maxFte);
    // Round up to nearest 0.5 FTE (you don't hire 2.3 teachers)
    return Math.ceil(computed * 2) / 2;
  }

  // Fixed mode — use the static fte value
  return r.fte;
}

Update computePersonnelForYear() (line 605) to accept enrollment and use the new helper:

typescriptexport function computePersonnelForYear(
  rows: StaffingRow[], salaryEsc: number, prorationFactor: number, y: number,
  enrollment?: number  // NEW parameter, optional for backward compat
): number {
  let total = 0;
  for (const r of rows) {
    const effectiveFte = enrollment !== undefined
      ? computeEffectiveFte(r, y, enrollment)
      : r.fte;
    const annual = effectiveFte * r.annualizedRate;
    const isContractNoPL = r.employmentType === "contract" && !r.payrollLike;
    let benefits = 0, tax = 0;
    if (!isContractNoPL) {
      if (r.benefitsEligible) benefits = annual * (r.benefitsRate / 100);
      tax = annual * (r.payrollTaxRate / 100);
    }
    total += annual + benefits + tax;
  }
  const esc = Math.pow(1 + salaryEsc, y);
  const pf = y === 0 ? prorationFactor : 1;
  return total * esc * pf;
}
